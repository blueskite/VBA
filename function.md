# プログラムが大きくなってきたら
1つのマクロ関数(Sub) で記入しているとプログラムが長くなり見通しが悪くなります。
このときバグを入れ込んだり、テストをするのが大変となるのでFunction関数や複数の標準モジュールを利用して見通し良いプログラムを作成することができます。


## Function
Function関数は一定の処理をひとまとめにしたプログラムです。
長いプログラムを複数の関数に分けて見通しをよくしたり、一定の処理を何回も実行するときに有効です。
